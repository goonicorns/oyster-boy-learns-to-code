Hint for Exercise 01: Variables

The variable declarations are almost there — just uncomment them and fill in the values.

For the long form:
  var name string = "Alice"

For the short form (most common in Go):
  city := "Tokyo"
  isSunny := true

The := operator means "create this variable and give it this value."
Go will figure out the type from what you put on the right side.
"Tokyo" is a string. 28 is an int. true is a bool.
